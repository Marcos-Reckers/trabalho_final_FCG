main.cpp: Contém apenas a função main e referências a funções/módulos externos.

headers/: Pasta para arquivos de cabeçalho (.h).
    utils.h: Declaração de funções utilitárias.
    shaders.h: Declaração das funções relacionadas ao carregamento de shaders.
    callbacks.h: Declaração das funções de callback.
    text_rendering.h: Declaração das funções relacionadas à renderização de texto.

sources/: Pasta para arquivos de implementação (.cpp).
    utils.cpp: Implementação de funções utilitárias.
    shaders.cpp: Implementação de funções relacionadas ao carregamento de shaders.
    callbacks.cpp: Implementação de funções de callback.
    text_rendering.cpp: Implementação de funções relacionadas à renderização de texto.
